# 💡 Exercise 01: The Light Switch

## Descripción

En este ejercicio aprenderás a crear una interfaz interactiva. El objetivo es simular un interruptor de luz: al hacer clic en un botón, el color de fondo debe cambiar y el texto debe actualizarse para reflejar si la luz está encendida o apagada.

## ¿Qué conceptos vas a practicar?

* **`useState`**: Crear variables de estado que refrescan la pantalla.
* **Event Handling**: Capturar clics mediante `onClick`.
* **Lógica Condicional**: Usar el operador ternario para cambiar estilos CSS.
