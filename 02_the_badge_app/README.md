# 📛 Exercise 02: The Badge App

## Descripción
¡Vamos a crear un generador de identificaciones en tiempo real! El objetivo es capturar lo que el usuario escribe en un campo de texto (input) y mostrarlo instantáneamente dentro de una tarjeta de diseño.

## ¿Qué conceptos vas a practicar?

* **Controlled Components**: Vincular el valor de un input al estado de React.
* **Event Object**: Acceder a la información del teclado mediante `event.target.value`.
* **Strings dinámicos**: Renderizar texto variable dentro de JSX.
